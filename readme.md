# Marknote

* Note taking app in markdown
* Jekyl inspired simplicity
* iPhone, iPad, iApp
		* Later: text editor, web, url app (turns html to md then saves)
* Syncs across devices
* Syncs to Evernote (later github, dropbox)
* Manage notes by
		- Title
		- Tags
		- Date created/edited
		- Location written/edited
